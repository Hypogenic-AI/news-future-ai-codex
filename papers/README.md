# Downloaded Papers

1. Are LLMs Prescient? A Continuous Evaluation using Daily News as the Oracle
   - Authors: Hui Dai, Ryan Teehan, Mengye Ren
   - Year: 2025
   - Venue: ICML 2025 (PMLR 267)
   - File: 2025_dai_daily_oracle.pdf
   - Why relevant: Introduces Daily Oracle, a continuously updated forecasting benchmark built from daily news; directly evaluates LLM temporal generalization and forecasting under closed/open-book settings.

2. OpenForecast: A Large-Scale Open-Ended Event Forecasting Dataset
   - Authors: Zhen Wang, Xi Zhou, Yating Yang, Bo Ma, Lei Wang, Rui Dong, Azmat Anwar
   - Year: 2025
   - Venue: COLING 2025
   - File: 2025_openforecast_coling.pdf
   - Why relevant: Provides a large-scale open-ended event forecasting dataset and tasks; includes LLM-based evaluation for open-ended forecasts.
